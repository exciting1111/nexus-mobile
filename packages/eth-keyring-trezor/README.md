# `@rabby-wallet/eth-keyring-trezor`

A Ethereum key management library designed for handling trezor Addresses. It enables effortless addition, removal, and management of trezor Addresses, offering a suite of methods to handle operations associated with these addresses, such as serialization, deserialization, and data normalization.

## Installation

`yarn add @rabby-wallet/eth-keyring-trezor`

or

`npm install @rabby-wallet/eth-keyring-trezor`
